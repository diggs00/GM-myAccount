Buttons

Button elements, links with a button class, and input elements should all render exactly the same.

<button>Button</button>
<input type="submit" value="Submit">
<input type="button" value="Input">
<a class="button">Link</a>
Link

Secondary Buttons

Adding a secondary-button class will display a secondary color.

<button class="secondary-button">Button</button>
<input type="submit" value="Submit" class="secondary-button">	q
<input type="button" value="Input" class="secondary-button">
<a class="button secondary-button">Link</a>
Link

Disabled Buttons

Adding a disabled-button class will create full-width, block level buttons. Button styles can be combined.

<button class="disabled-button">Button</button>
<input type="submit" value="Submit" class="disabled-button">
<input type="button" value="Input" class="disabled-button">
<a class="button disabled-button">Link</a>
Link

Full Width Buttons

Adding a full-button class will create full-width, block level buttons. Button styles can be combined.

<button class="full-button">Button</button>
<input type="submit" value="Submit" class="full-button">
<input type="button" value="Input" class="full-button">
<a class="button full-button">Link</a>
Link

Group Buttons

Adding a group-button-container class to a containing div will allow 2 buttons to sit next to each other.

<div class="group-button-container">
	<button class="secondary-button">Button</button>
	<button>Button</button>
</div>

Forms

Default forms will have full-width, block level labels, inputs, selects, and textareas.

<form>
	<div class="form-group">
		<label for="">First Name</label>
		<input type="text" id="" name="">

		<label class="myAccount-form-checkbox">Remember Username
			<input type="checkbox" id="" name="" value="">
			<span class="myAccount-checkmark"></span>
		</label>

		<label for="">Select</label>
		<select id="" name="">
			<option value="option-1">Option 1</option>
			<option value="option-2">Option 2</option>
		</select>

		<label for="">Password</label>
		<div class="myAccount-password">
			<input type="password" id="" class="input-myAccount-password gmocPasswordField" password-shown="false">
			<button class="secondary-button" type="button">Show</button>
			<small><a href="">Forgot username or Password?</a></small>
			<div class="password-meter">
				<div class="password-meter-bar password-meter-bar-danger" role="progressbar" aria-valuenow="0" aria-valuemin="0" aria-valuemax="100"></div>
				<div class="password-meter-body"</div>
			</div>

		<ul class="myAccount-password-instruction">
			<li>
				<p>Your password MUST include:</p>
			</li>
			<li class="success">
				<div class="check-mark__wrapper">
					<div class="check-mark"></div>
				</div>
				<p>1 number</p>
			</li>
			<li class="fail">
				<div class="check-mark__wrapper">
					<div class="check-mark"></div>
				</div>
				<p>8 characters in length</p>
			</li>
			<li>
				<p>Note: Passwords are case sensitive</p>
			</li>
		</ul>

		<input type="submit" value="Log In">

	</div>
</form>
Forgot username or Password?

  • Your password MUST include:

  • 1 number

  • 8 characters in length

  • Note: Passwords are case sensitive.
